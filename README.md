## 🔧 Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS, `tailwindcss-animate`, `tailwind-merge`, `clsx`
- **UI Components**: Radix UI, Lucide, Vaul
- **Forms & Validation**: React Hook Form, Zod, @hookform/resolvers
- **State Management**: React State / Form State
- **Date Utility**: `date-fns`
- **Charts**: Recharts
- **Carousel**: `embla-carousel-react`
- **OTP Input**: `input-otp`
- **Misc**: `cmdk`, `sonner`, `react-resizable-panels`, `next-themes`, `class-variance-authority`

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/my-project.git
cd my-project
```

Install dependencies:

```bash
npm install --force
# or
yarn install
```

---

## 🚀 Usage

### Development

```bash
npm run dev
# or
yarn dev
```

### Production

Build for production:

```bash
npm run build
```

Start production server:

```bash
npm start
```

### Linting

Run ESLint:

```bash
npm run lint
```
